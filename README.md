

# TYPERPRO 🎯

Simple tool to auto-press selected keys with custom timing ⏳.  
Save and load key setups 📂, press in endless loops ♻️.

---

## Current ✅:
- Add/remove keys ➕➖  
- Set press durations 🕒  
- Start/Stop pressing 🔥  
- Save/Load settings 💾  

---
